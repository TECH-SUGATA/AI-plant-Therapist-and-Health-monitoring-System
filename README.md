🌿 AI Plant Therapist & Health Monitoring

An AI-powered plant health monitoring system that helps detect plant diseases, analyze plant health conditions, and suggest treatments.
This project uses Artificial Intelligence, Computer Vision, and modern web technologies to assist farmers, gardeners, and plant lovers in maintaining healthy plants.

🧠 How It Works

1️⃣ User uploads a plant leaf image
2️⃣ AI model analyzes the leaf patterns and symptoms
3️⃣ System identifies plant disease or health condition
4️⃣ Platform suggests treatment and care methods
5️⃣ User can monitor plant health over time

⚙️ Installation & Setup

You can use any IDE of your choice.

Requirements

Node.js

npm

Install Node using nvm
https://github.com/nvm-sh/nvm#installing-and-updating

Step 1 — Clone the Repository
git clone <YOUR_GIT_URL>

Step 2 — Navigate to the Project Folder
cd ai-plant-therapist 

Step 3 — Install Dependencies
npm 

Step 4 — Run Development Server
npm run dev

Your project will start with auto reload and instant preview.
